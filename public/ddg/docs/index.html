NOTE: There are 3 ways we can include the get-min.js
             		                            
             		           
                               
                            
                 ¶               
               
    

  • include the script in the <head> after YUI with a generic <script> tag
    • 

  • use YUILoader w/callback as in YAHOO.util.YUILoader to load ['get'] module
    • 

  • append <script> to <body>, after DOM is ready using readyState (currently used)
    •
